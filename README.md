# Canton Dog Walking Club Demo Website

Static demo website for `cantondogwalking.club`.

## Files

- `index.html` — homepage
- `services.html` — services and sample pricing
- `about.html` — about/trust page
- `contact.html` — contact/request page
- `assets/style.css` — styling
- `assets/logo.svg` — demo logo
- `assets/hero-dog.svg` — demo hero illustration

## GitHub Pages

Upload all files/folders to a repo. Go to Settings > Pages. Deploy from the `main` branch and `/root`. Add `cantondogwalking.club` after the domain is purchased/configured.

## Customize before showing a real client

Replace business name, email, service area, pricing, reviews/testimonials, real photos, and any policies.

This is intentionally a starter demo, not a perfect final business website.
